# Cium Pet Backend.

This is a simple pet helpfully application powered by Nest JS.

## How to run?

- `npm install`
- `npm run db:connect`
- `npm run start:dev`

## Services.

- ### Authentications.

  - [x] Join.
  - [x] Login.
  - [x] JWT access token.
  - [x] JWT refresh token.
  - [x] Logout.

- ### Users.

  - [ ] Me.
  - [ ] Profile modify.

- ### Products.
  - [ ] List.
  - [ ] Create.
  - [ ] Retrieve.
  - [ ] Delete.
  - [ ] Modify.
